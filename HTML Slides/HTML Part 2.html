Links // How To

Hyperlinks

The power of the web is that pages can lead to other pages. When you click on a link on a web page, it takes you to another page. This link is called a hyperlink.

Hyperlinks are created with anchor elements, which generally look like:

<a href="https://www.google.de">Google</a>

and render on the page like this: Google.

Inside the opening a tag there is href, which stands for "reference." This is called an attribute. Attributes like href describe the properties of HTML elements. In this case, the href attribute is the target URL that the link will open. The content inside the anchor element is the text that users see displayed on the page.

This is the format that you must use when you make hyperlinks! Note:

  • There is a space between a and href.
  • There are no spaces around the =.
  • The website has two " around it.
  • There are no spaces between the href attribute and the > of the opening tag.

HTML Form - Form Processing

Anatomy of a form

  • An input tag (no close tag!)
    
        <input>
  • A type attribute 
    
        <input type="checkbox">
        <input type="radio">
        <input type="text">
  • A name attribute (the value belongs to which field)
    
        <input name="agree-terms-and-conditions" type="checkbox">
        <input name="areas-of-interest" type="radio">
        <input name="street-address" type="text">
  • Human readable labels
    
        <input name="agree-terms-and-conditions" type="checkbox">I agree with terms and conditions
        <input name="areas-of-interest" type="radio">I like music
        <input name="street-address" type="text">Kurfürstendamm
  • Has a submit input field
    
        <input name="agree-terms-and-conditions" type="checkbox">I agree with terms and conditions
        <input name="areas-of-interest" type="radio">I like music
        <input name="street-address" type="text">Kurfürstendamm
        <input type="submit">Buy!
  • Input fields are surrounded by form tag
    
        <form>
          <input name="agree-terms-and-conditions" type="checkbox">I agree with terms and conditions
          <input name="areas-of-interest" type="radio">I like music
          <input name="street-address" type="text">Kurfürstendamm
          <input type="submit">Buy!
        </form>
  • Form has action (url) and method attribute (get or post)
    
        <form action="http://myshop.com/order.php" method="post">
          <input name="agree-terms-and-conditions" type="checkbox">I agree with terms and conditions
          <input name="areas-of-interest" type="radio">I like music
          <input name="street-address" type="text">Kurfürstendamm
          <input type="submit">Buy!
        </form>
